# Ingot
## an unblocker that works by removing the blocking extension
### run the script by going the website linked below or following the instructions below.


[Lauch ingot](https://mmarchjr.github.io/Ingot2022)


### to use it as a bookmark, copy the code below and paste it in your boomark bar:
```
javascript: (function () {
    var a = document.createElement("script");
    a.src = "https://cdn.jsdelivr.net/gh/mmarchjr/unblock-anything-not-proxy-@latest/Ingot-src.min.js";
    document.body.appendChild(a);
})();
```
